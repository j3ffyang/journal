A personal journal covering open-source technology, Linux, motorcycle riding, photography, and travel around the world.

**Live site:** [everbox.io](https://everbox.io)

## Topics

- **Technology** — open-source software, Linux, systems, AI/ML
- **Motorcycle** — riding experiences and trips
- **Photography** — travel and street photography
- **Travel** — destinations around the world

## Stack

Built with [AstroPaper](https://github.com/satnaing/astro-paper) theme on [Astro](https://astro.build), deployed via GitHub + Cloudflare Pages.

## Local Development

```bash
pnpm install
pnpm run dev        # http://localhost:4321
pnpm run build      # production build + pagefind index
pnpm run preview    # preview production build
```
